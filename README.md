# new-post-sender-public
## Получить данные отправителя Новой почты

Обработка позволяет по api-ключу "Новой почты" узнать всех текущих зарегистрированых отправителей и их контактные лица.

1. Ввести api-ключ.
2. Нажать "Выполнить"
